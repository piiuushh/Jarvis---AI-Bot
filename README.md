# Jarvis---AI-Bot

Jarvis---AI-Bot is an intelligent virtual assistant designed to interact with users via conversational AI. Built using Python, Jarvis can answer questions, execute commands, and automate tasks on your machine.

## Features

- Natural language processing for understanding user requests
- Executes system commands and automates common workflows
- Answers general knowledge questions
- Extensible to add more skills and custom tasks

## Technologies Used

- Python
- Machine Learning & NLP (using libraries like `SpeechRecognition`, `pyttsx3`, `wikipedia`)
- OS and system libraries

## Getting Started

### Prerequisites

- Python 3.7+
- pip

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/piiuushh/Jarvis---AI-Bot.git
    cd Jarvis---AI-Bot
    ```

2. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

### Usage

1. Run the main script:

    ```bash
    python jarvis.py
    ```

2. Speak or type commands to interact with Jarvis.

## Example Commands

- "What's the weather today?"
- "Open notepad"
- "Search Wikipedia for Python programming"
- "Play music"

## Extending Jarvis

You can add new skills or commands by editing the core files and following the structure for intent recognition and response handling.
